## Build and deploy a Web App with Flask using EKS

Criar e disponibilizar um simples Web App com Python e Flask usando EKS

## Ferramentas utilizadas

* Python
* Flask
* Docker
* Amazon Elastic Kubernetes Service (Amazon EKS)
